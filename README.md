VNB BANKING SYSTEM
===================

VELELO NATIONAL BANK - Digital Banking Solution

FEATURES:
---------
1. Customer Account Management
2. Secure PIN-based Authentication
3. Deposit/Withdrawal/Transfer Operations
4. Transaction History
5. Account Information Display
6. Beautiful Console Interface with VNB Branding

HOW TO COMPILE:
---------------
Windows: Double-click compile.bat
Linux/Mac: g++ -std=c++17 *.cpp -o VNBBank

REQUIREMENTS:
-------------
- C++17 Compiler (g++ or MSVC)
- Windows/Linux/Mac with ANSI support

FILES:
------
main.cpp       - Main application
VNBBank.h/cpp  - Banking system
Account.h/cpp  - Account management
Transaction.h/cpp - Transaction handling
VNBUI.h/cpp    - User interface
compile.bat    - Windows compilation script

USAGE:
------
1. Run the program
2. Choose from main menu:
   - Login with account number and PIN
   - Create new account
   - View banking services
   - Access account dashboard

SECURITY:
---------
- 4-digit PIN authentication
- Account number masking
- Secure input for sensitive data
- Transaction logging
